
> 使用统一参数编译源码文件，不支持为一些模块定制编译参数，适合一些中小型的项目工程。

1、在 [shell] 中进入 [Makefile] 所在的目录

```bash
cd 2_medium   # 进入[Makefile]所在的目录
```

2、初次使用时，重置编译状态，同时创建所需文件夹

```bash
make clean    # 执行清除操作（同时创建所需文件夹）
```

3、直接编译示范工程（清除操作之后会重新编译所有）

```bash
make all      # 执行编译操作
```
